# RWA Apartment Model

This repository contains an experimental model for filtering and scoring small and medium-sized apartments in Korea and the US, with a focus on Real World Asset (RWA) use cases.

- Collect and clean basic apartment market data (Korea + US)
- Filter units by size (around 60㎡ and equivalent US small units), price, and rental yield
- Apply simple scoring rules to identify attractive properties for potential RWA tokenization

The goal is not to be a full production system, but a practical playground where I iterate on data pipelines, filters, and scoring logic that can later be plugged into on-chain RWA products.
